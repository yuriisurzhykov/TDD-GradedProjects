Palindrome - is a word that could be read from left to right and from right to left similar way.
For example: Madam.

1. The application should have input field and button to check if entered word is palindrome.
2. User should be able to input sentence with punctuation mark.
3. Punctuation mark must not be counted while checking.
4. If user click on button with empty string, there will be an input error under the input field.
5. If the entered word or sentence is palindrome then OK emoji should appears.
6. If the word/sentence is not a palindrome - dislike emoji should appears.
